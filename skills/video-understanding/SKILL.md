# Video Understanding

帮你理解视频内容的技能。

## 支持的平台

- **B站 (bilibili)**: 视频描述、标题、UP主信息
- **YouTube**: 视频标题、描述、字幕（如果有）
- **其他网页视频**: 基本元信息

## 工作流程

### 1. 获取视频基本信息

使用 `web_fetch` 工具获取视频页面内容：

```bash
# B站视频
web_fetch url="https://www.bilibili.com/video/BVxxxxx"

# YouTube
web_fetch url="https://youtube.com/watch?v=xxxxx"
```

这会提取视频的标题、描述、发布时间等元信息。

### 2. 尝试获取字幕（仅YouTube）

对于YouTube视频，可以尝试提取字幕：
- 使用 `browser` 工具播放视频并打开字幕
- 或者告知用户当前技能无法直接获取字幕，建议用户手动复制字幕内容

### 3. 总结内容

根据获取的信息，用自己的话总结视频：
- 视频主题是什么
- 主要内容/观点
- 视频来源和可信度

## 视频信息来源

通过 web_fetch 可以获取的内容：
- 视频标题
- 视频描述/简介
- UP主/频道信息
- 发布时间
- 播放量/观看次数（部分网站）

## 限制

- 无法直接播放视频音频
- B站等国内平台的字幕获取较困难
- 某些视频可能需要登录才能查看完整信息

## 触发方式

当用户发送视频链接或询问"这个视频讲了什么"时自动触发。
